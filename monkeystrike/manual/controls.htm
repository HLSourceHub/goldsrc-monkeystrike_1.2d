Controls And Commands


Here are all commands listed you can bind keys to and which are for use in Monkeystrike:


  • Doublejump - If you press the jump key when you're already in the air, you will jump again one time.


  • Fly (Hold key) - Activates the backpack of the lemur (press this key as long as you want to fly).


  • Fly (Toggle key) - Activates the backpack of the lemur (press this key one time for activating it, one time for deactivating it).


  • Toggle Thirdperson mode - Switches between Thirdperson and Firstperson view.


  • Radio Command Menu - Brings up the radio command menu.


  • Enable Background Music - Activates the background music track (tracks not available in this release!).


  • Disable Background Music - Disables the background music track (tracks not available in this release!).


  • Change Team - Brings up the menu where you can choose a team.


  • Change Class - Brings up the menu where you can choose a monkey class.
